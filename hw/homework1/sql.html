The language used to interact with relational databases is SQL (Structured Query Language). With SQL tables can be created, altered, and deleted. Records can be inserted, updated, and selected from the tables for viewing. Each SQL query specifies what is to be done to which table.

Examples of some SQL statements:

Statement Result
SELECT * FROM customers; Returns all records and fields from the table titled customers.
SELECT fname, lname FROM customers WHERE age > 30; Returns the first and last name of customers older than 30.
INSERT INTO customer (fname, lname, age) VALUES ('Bob', 'Dole', 95); Inserts a customers record into the customer table.
UPDATE customers SET fname='Steve'; Updates all customers first names to Steve.
DELETE FROM customer WHERE lname='Dole'; Deletes all records from customer table where the the customers last name is Dole.
