Database - a comprehensive collection of related data organized for convenient access, generally in a computer.

Databases, put simply, are a collection of data held in an organized way. There are multiple ways to organized data to store, each with its strengths and weakneses. Realational Databases are databases that organize data into multiple tables that are linked to other tables through the use of 'keys', or common bits of information in separate tables.
Each table is made up of rows and columns, representing the records, and the fields that the table holds.

Some of the more popular Relational Databases in use today are:

  • Oracle
  • MySQL
  • Microsoft SQL Server
  • PostgreSQL
  • Microsoft Access
